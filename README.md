# yii2-valet-driver
Yii2 Valet Driver

> install 

```
cd ~/.config/valet/Drivers/
wget https://raw.githubusercontent.com/chinaphp/yii2-valet-driver/master/Yii2ValetDriver.php
```

## yii2-app-basic

```
valet link app-name
http://app-name.test
```

## yii2-app-advanced

```
cd backend
valet link backend-app-name
http://backend-app-name.test
```
